# Julfar Consultancy Services-Task

Create a simple React application that consumes an API to fetch users information based on gender (Male, Female or Both) 
 
## Getting Started

These instructions will get you a copy of the project up and running on your local machine for testing purposes. 

### Local Environment  

### Installing 
1. Clone the repository to your local machine: `git clone https://github.com/ahmedosama704/Julfar-Consultancy-Services.git` 
2. Navigate into the cloned repository: `cd jsc-task` 
3. Install all dependencies: `npm install` or `yarn` 

#### You will have to run the website
1. Run the website: `yarn start || npm run start`
